PID_calculator instalation instructions
Author: jptindade3
April, 2020

To install this librarie in your Arduino IDE, you'll need to: 
1 - Grab the "PID_calculator" folder, copy it;
2 - Paste it inside the folder "libraries", that will be located inside your Arduino instalation folder;
3 - Once you've done it, close your Arduino IDE, if it's already opened, and open it again, then you should be able to find "PID_calculator" between all the other libraries on your IDE. The sample codes should be avaluable as well.

The avaluable classes, functions and variables are in the "PID_calculator.h" file, and further explanation about it are in the "PID_calculator.cpp" file.
